🔗 URL Shortener

A full-stack URL Shortener application that allows users to convert long URLs into short, shareable links and track their usage with analytics.

🚀 Features
🔐 User Authentication (JWT-based login/signup)
✂️ Shorten long URLs instantly
📊 Track click analytics (date-wise / total clicks)
📋 View all shortened URLs in dashboard
📈 Graph visualization for clicks
🌐 Responsive UI
🛠️ Tech Stack
Frontend
React.js
Tailwind CSS
TanStack React Query (v5)
Backend
Spring Boot
Spring Security + JWT
REST APIs
Database: MySQL / PostgreSQL (depending on your setup)
